 \item Define $\gcd(a,b,c)$ to be the largest integer $d$ such that $d|a$, $d|b$, and $d|c$.  Let $a$, $b$, and $c$ be integers such that $\gcd(a,b,c)=1$. Prove that there are integers $w$, $x$, and $y$ such that $aw+bx+cy=1$.  (Note that it is possible in this problem to have $\gcd(a,b)>1$, $\gcd(b,c)>1$, and $\gcd(a,c)>1$.) (Hint: Use Bezout's identity)
 
\item Without explicitly squaring any integers modulo $m$ for any integer $m$, determine if $15$ is a square modulo $77$. 

 \item Fix a prime number $p$ and an integer $n\not\equiv 0,1\pmod{p}$. Set $x=n^{p-2}+n^{p-3}+\cdots+n+1$. Prove that $p\mid x$. (Hint: consider expanding $(n-1)x$.)

\item(Peer review problem.)  Show that there are no integers $x$, $y$, and $z$ such that $x^3+y^3+7z^4=3$. 

\item Prove that, if $a$ is a primitive root modulo $m$, then $a$ must be a unit modulo $m$.
